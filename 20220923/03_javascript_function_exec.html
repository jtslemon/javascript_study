문. 함수를 활용한 비만도 측정지수 구하기

  1. 신장을 입력하기 위한 명령프롬프트와 체중을 입력하기 위한 명령프롬포트를 사용
  2. 함수에 신장, 체중값을 사용하여 계산식 작성하기
  3. 적정체중인지를 비교하여 결과를 출력한다.

BMI지수 = 몸무게(kg) / (신장(m) x 신장(m))

BMI지수가 18.5 이하면 저체중, 18.5~23은 정상, 23~25는 과체중, 25~30은 비만, 30~35는 고도비만

저체중은 배경색 주황색, 정상은 녹색, 과체중은 파랑, 비만은 보라색, 고도비만은 빨상색 배경으로 서식을 지정함.
